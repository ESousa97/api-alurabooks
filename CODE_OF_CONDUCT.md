# Código de Conduta

Este projeto adota o Contributor Covenant, versão 2.1.

## Nosso compromisso

Contribuímos para manter uma comunidade acolhedora e respeitosa para todos.

## Padrões de comportamento

**Exemplos de comportamento que contribuem para um ambiente positivo:**

- Empatia e respeito com outras pessoas.
- Críticas construtivas e objetivas.
- Responsabilidade por palavras e ações.

**Exemplos de comportamento inaceitável:**

- Linguagem ofensiva ou ataques pessoais.
- Assédio, intimidação ou discriminação.
- Exposição de informações privadas sem consentimento.

## Aplicação

Casos de abuso podem ser reportados abrindo uma issue ou entrando em contato com os mantenedores.

## Atribuição

Este Código de Conduta é baseado no [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct.html).
