# Changelog

Todos os registros importantes deste projeto serão documentados neste arquivo.

O formato segue [Keep a Changelog](https://keepachangelog.com/pt-BR/1.1.0/) e este projeto adota [SemVer](https://semver.org/lang/pt-BR/).

## [1.1.0] - 2026-01-14

### Added
- Estrutura de pastas padronizada com `src/`, `tests/` e `docs/`.
- Lint e formatação com ESLint e Prettier.
- Testes básicos com Jest e Supertest.
- CI com GitHub Actions e CodeQL.
- Dependabot para npm e GitHub Actions.
- Documentação adicional (contribuição, segurança, código de conduta).

### Changed
- Servidor movido para `src/server.js`.
- Uso de variáveis de ambiente para segredos e CORS.

### Removed
- Certificados locais não utilizados.
